# Proyecto LDU Quito - Motor de Renderizado Pug

Página web sobre Liga Deportiva Universitaria de Quito desarrollada con el motor de renderizado Pug.

## Tecnologías

- Node.js
- Express
- Pug (Motor de renderizado)
- CSS3

## Características

- **Inicio**: Información general del club con imágenes
- **Historia**: Línea de tiempo con momentos importantes
- **Jugadores**: Plantel actual y leyendas históricas
- **Palmarés**: Títulos nacionales e internacionales
